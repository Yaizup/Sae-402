# Site web sur le Parthénon pour la SAE 402
- contient une carte permettant de localiser le Parthénon
- contient un modèle 3D du Parthénon, d'une lance et d'un bouclier
- contient un articles en français et en anglais sur l'histoire du Parthénon

# Tutoriel d'installation

## Prerequis :
- node.js (https://nodejs.org/en/download/)

1. Installer les dépendances : `npm install`
2. Lancer le projet : `npm run dev`
3. Accéder à l'application : `http://localhost:5173`

## ou si node est déjà installé :
```bash
npm install && npm run dev
```
